#  issue distribution charts

Draw issue distribution charts with antv/g2[ (Docs) ](https://antv.vision/zh).

Figures are in the public folder.


## Usage

### Requirements

- NodeJS
- npm

### How to Start

```
npm install
npm start
```

Then Visit http://localhost:8000/.
